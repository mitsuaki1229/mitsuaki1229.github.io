# mitsuaki1229.github.io

[mitsuaki1229.com](https://mitsuaki1229.com) website.

## Requirement

* [Bundler](https://bundler.io/)
* [Jekyll](https://jekyllrb.com/)

## Install

```shell
$ cd docs/
$ bundle install
```

## Development

```shell
$ cd docs/
$ bundle exec jekyll serve
```
